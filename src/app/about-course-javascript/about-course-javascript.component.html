Ծրագրավորման հիմունքներ, Տևողությունը՝ 1 ամիս,
HTML / CSS Տևողությունը՝ 2 ամիս.
JavaScript Տևողությունը՝ 6 ամիս,
Angular Framework Տևողությունը՝ 3 ամիս,
NodeJS/MongoDB Տևողությունը՝ 2 ամիս

Արժեքը 30,000 դրամ / 1 ամիս

JavaScript full-stack վեբ ծրագրավորման դասընթացն անցկացվում է մոդուլային ձևաչափով: Յուրաքանչյուր մոդուլից հետո ուսանողները հանձնում են քննություն և, կախված արդյունքներից, հնարավորություն են ստանում շարունակել ուսումը հաջորդ մոդուլներում։ Full stack դասընթացի ավարտից հետո ուսանողները ստանում են վկայականներ, իսկ ավելի բարձր միավորներ և առաջադիմություն ունեցողները հնարավորություն են ստանում պրակտիկա անցնել Brain Fors ընկերությունում և ստանալ 50% փոխհատուցում ամբողջ դասընթացի համար:
Պրակտիկայի ընթացքում լավագույն արդյունք ցույց տված մասնակիցները կստանան աշխատանքի առաջարկ։
